Реализован метод autocomplete следующим образом. Получаем введенные пользователем символы(префикс). По нажатию клавиши TAB дополняем слово, если слово является префиксом другого, после повторного нажатия дополняется следующим словом. 
пример r -> TAB -> run -> TAB -> runner. Слова можно вводить как через пробел так и после нажатия enter, автодополнение будет работать.
